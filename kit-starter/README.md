# kit-starter

Materiale practice pentru training-ul de web dev — BEST Iasi.

## Cele 3 demo-uri

| Folder | Cand il folosesti | Ce de observat |
|---|---|---|
| `00-blank/` | Pornesti un site nou de la 0 | Schelet minim: viewport, defer, variabile CSS |
| `01-broken/` | Inveti sa identifici probleme cu Lighthouse | 10 probleme deliberate (title, viewport, alt, contrast, defer, heading order...) |
| `02-reference/` | Vezi cum trebuie sa arate finalul | Best practices aplicate complet (Lighthouse 95+/100) |

## Workflow recomandat

1. **Ruleaza** `02-reference/` cu Live Server. Click prin el. Vezi cum arata target-ul.
2. **Observa** structura. Citeste `02-reference/README.md` — fiecare best practice e explicata.
3. **Practica** in `00-blank/` — pornesti de la 0, construiesti site-ul tau.
4. **Audit** in `01-broken/` — ruleaza Lighthouse, vezi ce gaseste, fixeaza.

## In timpul training-ului

Fiecare demo se mapeaza pe o faza din `pdf/codex.pdf`:

| Demo | Faza Codex |
|---|---|
| `02-reference/` | §1-§5 (cod complet adnotat) |
